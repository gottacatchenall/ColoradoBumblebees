`01_prerequisites`

This folded contains scripts that need to be run to in order to download
required data and preprocess other data into a usable form by the rest of the
workflow. 

- `chelsa` downloads CHELSA and computes projections for each decade for each
  SSP
- `gbif_query` submits a query to GBIF. This requires GBIF credentials.
- `phylogeny` contains assets raw data to usable in `BEAST v2` and to infer phylogenies and
  visualize them.

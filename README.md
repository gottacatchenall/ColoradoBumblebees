# ColoradoBumblebees

This repository contains the code associated with the manuscript _Projecting the
rewiring on a plant-pollinator network across the southern Rocky Mountains_
(Catchen et al. _in prep_). 

Each script in the `scripts` folder goes through the process to reproduce the
analysis from the manuscript. 

> **Warning**
> Some of the data required to run the analysis are embargoed, and therefore
> will not be on Github until the paper is published. If you are on the MOU for
> this data and would like to run the code, email me. 

All analysis was done using `julia v1.8.5`. In addition, special branches with
particular work-around have to be used for two packages.
